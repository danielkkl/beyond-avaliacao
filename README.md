# beyond-avaliacao
Ficha de Avaliação Fisioterapêutica SaaS

## Deploy em servidor IONOS

Veja o guia completo em [`DEPLOY_IONOS.md`](./DEPLOY_IONOS.md).

## Empacotamento para WinRAR

Use o guia [`PACOTE_WINRAR_PRONTO.md`](./PACOTE_WINRAR_PRONTO.md) e rode `scripts\create_release_bundle.bat` no Windows.

## Baixar pacote

Para gerar e baixar rapidamente, veja [`COMO_BAIXAR_PACOTE.md`](./COMO_BAIXAR_PACOTE.md).

## Pronto para rodar

Use [`PRONTO_PARA_RODAR.md`](./PRONTO_PARA_RODAR.md) e execute `scripts\pronto_para_rodar.bat` (Windows) ou `./scripts/pronto_para_rodar.sh` (Linux/macOS).
